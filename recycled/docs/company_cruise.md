公司视图 — 考察链渐进式探索

  核心概念是考察链（Investigation Chain）：一种路径导向的公司关系渐进探索方式，不是一次性加载全局网络，而是沿着上下游逐步深入。

  ────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────

  1. 显示层级

   层级           显示方式           行为

  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   当前考察节点   实体、高亮、中心   唯一焦点

   考察链节点     实体、非高亮       从起点到焦点的路径节点，保留历史

   Preview 节点   半透明、虚线边框   当前焦点的上下游，切换焦点后消失

   固定节点       实体（双击产生）   永久保留，不受回退影响

  2. 交互规则

  • 单击 preview 节点 → 该节点加入考察链，成为新焦点；旧 preview 全部消失；加载新 preview

  • 单击考察链旧节点 → 回退截断到该节点，其后的所有节点和 preview 消失

  • 双击当前焦点 → 将其所有 preview 节点永久固定（不再随焦点切换消失）

  • 全局模式下单击 → 高亮该节点，dim 无关节点

  • 全局模式下双击 → 以该节点为起点启动局部考察

  3. 布局要求

  • 当前考察节点缓慢移动到画面中心，其它节点相对位置保持不变。
