## 一、任务概述

### 目标
基于Tushare A股数据，通过手工分析每家公司的原材料（输入）和产出，构建：
1. **产业实体图**（Neo4j）：仅包含实体/服务节点及INDUSTRIAL_FLOW关系
2. **公司视图**（PostgreSQL）：公司与产业节点的暴露关系
3. **行业视图**（PostgreSQL）：行业作为节点过滤器与产业节点的映射

### 约束遵循
- ✅ 产业图中**不包含**公司、产业、人员、组织，仅表达实体/服务间关系
- ✅ 行业是**配置出来的节点过滤器**，不是图中的元素
- ✅ 公司是**独立存储**的，不是图中的元素
- ✅ 每个节点/边均经过**手工分析**和**网络核查**
- ✅ 不确定的信息通过网络搜索验证

## 三、产业链分析过程

### 3.1 分析方法
对每家公司执行以下步骤：
1. **读取Tushare数据**：获取main_business、business_scope等字段
2. **网络搜索核查**：通过SearchWeb搜索公司官网、年报、行业报告
3. **提炼产业链**：
   - 上游（输入）：1-3个核心原材料/零部件/服务
   - 下游（产出）：1-3个核心产品/服务
4. **手工构造节点和边**：为每个实体定义node_id、definition、entity_type
5. **提交到系统**：通过GraphRegistrationBatch API提交

### 3.2 典型分析案例

#### 案例1：中国宝安（000009）— 新能源材料龙头

**信息来源**：
- Tushare main_business：高新技术、房地产、生物医药
- 网络核查：控股子公司贝特瑞是全球锂电池负极材料出货量第一（市场份额>20%）

**产业链构造**：
```
上游：天然石墨（natural_graphite）→ 自有鸡西石墨矿
      针状焦（needle_coke）→ 与山东京阳合资
      锂盐（lithium_salt）→ 市场化采购

中游：锂电池负极材料（lithium_battery_anode）→ 贝特瑞核心产出
      锂电池正极材料（lithium_battery_cathode）→ 高镍三元

下游客户：宁德时代、比亚迪、松下（间接特斯拉）
```

**关键发现**：
- 中国宝安是典型的"投资控股型"企业，真正的产业实体在子公司贝特瑞
- 上游天然石墨实现自有矿山供应，针状焦通过合资绑定
- 客户高度集中：全球前十大电池厂商中的绝大多数

#### 案例2：南玻A（000012）— 玻璃产业纵向一体化

**信息来源**：
- Tushare main_business：玻璃、光伏、电子玻璃
- 网络核查：拥有石英砂自有矿（安徽凤阳），光伏玻璃日熔量~9000吨

**产业链构造**：
```
上游：石英砂（quartz_sand）→ 自有矿（江油/凤阳/清远/北海）
      纯碱（soda_ash）→ 市场化采购
      天然气（natural_gas）→ 清洁能源燃料

中游：浮法玻璃（float_glass）
      光伏玻璃（pv_glass）→ 与隆基签订5年长单
      工程玻璃（engineering_glass）→ 央视大楼、大兴机场
      电子玻璃（electronic_glass）
```

**关键发现**：
- 南玻A的纵向一体化程度很高：从石英砂矿到玻璃成品到光伏电站
- 所有产线均使用天然气（国内同行唯一），环保优势明显
- 光伏玻璃是近年增长最快的业务板块

#### 案例3：国药一致（000028）— 医药流通商的本质

**信息来源**：
- Tushare main_business：医药、医药器械批发
- 网络核查：2024年营收734亿元，分销占比~70%，零售（国大药房）~30%

**产业链构造**：
```
上游：药品（pharmaceutical_product）→ 数千家供应商
      医疗器械（medical_device）→ 数百家供应商

中游：医药分销服务（pharmaceutical_distribution）
      医药零售服务（pharmaceutical_retail）→ 国大药房全国门店超8000家
```

**关键发现**：
- 国药一致**不生产药品**，本质是流通商，核心价值在渠道和物流网络
- 上游和下游均高度分散，前五供应商占比11-19%，前五客户占比不足5%
- 国大药房是中国连锁药店销售规模第一

#### 案例4：深科技（000021）— 国产存储封测核心

**信息来源**：
- Tushare main_business：磁头产品、硬盘芯片、内存产品
- 网络核查：2015年收购沛顿科技（原金士顿封测厂），合肥沛顿配套长鑫存储

**产业链构造**：
```
上游：存储晶圆（memory_wafer）→ 长鑫存储、长江存储、美光
      封装材料（packaging_material）→ 基板、塑封料等

中游：存储模组（memory_module）→ 内存条、SSD
      智能电表（smart_meter）→ 国家电网、意大利ENEL
```

**关键发现**：
- 沛顿科技是国内最大独立DRAM封测企业，承接长鑫存储超60%委外封测
- 是国内唯一具有Intel存储器测试验证合作资质的封测企业
- 为希捷全球唯一服务器硬盘头堆制造商

## 五、发现与启发

### 5.1 关于Tushare数据
1. **行业分类粒度粗**：Tushare的`industry`字段只有110个分类，且以证监会/申万行业为主，无法直接映射到产业链实体
2. **main_business字段质量参差**：部分公司的主营业务描述过于笼统（如"高新技术"），需要网络搜索补充
3. **缺少供应链数据**：Tushare不提供供应商/客户数据，产业链关系完全依赖外部搜索

### 5.2 关于产业链规律
1. **纵向一体化 vs 专业化分工**：
   - 南玻A（纵向一体化）：自有石英砂矿→玻璃→光伏电站
   - 深纺织A（专业化）：仅做偏光片，上游PVA/TAC依赖日本进口
2. **流通商 vs 制造商的本质差异**：
   - 国药一致（流通商）：不生产，核心价值在渠道
   - 中国宝安（制造商）：通过贝特瑞掌握负极材料核心技术
3. **客户集中度风险**：
   - 德赛电池：前五大客户占比65-74%，苹果单一客户>50%
   - 国药一致：前五客户占比不足5%，高度分散

### 5.3 关于系统设计
1. **EntityType的分类足够覆盖**：material/component/device/subsystem/service等分类能有效表达产业链层级
2. **INDUSTRIAL_FLOW的六种关系类型**：material_flow最常用（14/33），composition次之（8/33），能准确表达产业链关系
3. **Evidence机制很重要**：每个节点/边都需要证据支撑，迫使我们进行认真的网络核查

### 5.4 关于手工分析的价值
1. **脚本批量添加不可行**：每家公司都需要针对性的网络搜索和判断，无法自动化
2. **产业链关系需要领域知识**：如"针状焦→人造石墨负极"这种关系，不是通用知识库能覆盖的
3. **中国产业链的复杂性**：很多公司有"母公司-子公司"结构（如中国宝安-贝特瑞），需要穿透分析
